## Problema dei Lettori-scrittori 7.1.2

Problema: condividere un dile tra molti processi. ALcuni programmi devo scriverci sopra (scrittori) altri devono leggerlo (lettori)

Sutture deti condivise:

```
semaphore mutex = 1,scrivi = 1
int numerolettori = 0
```

Processo scrittore:

``` 
wait(scrivi);
...
scrivi sul file
...
signal(scrivi);
```

Codice del lettore:

```
wait(mutex);
numeroLettori++;
if(numeroLettori == 1) wait(scrivi);
signal(mutex);
...
leggi il file
...
wait(mutex);
numeroLettori--
if numeroLettori == 0 signal(scrivi)
signal(mutex)
```
