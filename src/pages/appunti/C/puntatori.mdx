# I puntatori

Un puntatore in C indirizza una variabile all'indirizzo di qualche byte.

Useremo due nuovi (simboli) `&` e `*`:

* `&` ritorna l'indirizzo della variabile
```c 
 int v = 25; // supponiamo che il suo indirizzo sia 8100
 int * p = &v; 
 // p = 8100 (l'indirizzo di v); *p = 25 in quanto v = 25 e le p "punta" a v
```
* `*` il puntatore si dichiarano i questo modo:

```c
 <type> * <nome>
```

esempio:

```c
 int v;
 int *p; //Inizializzo un puntatore

 v = 25;
 p = &v; // assegno l'indirizzo della variabile di v in p
```

## Array del puntatore

Definiamo un array:

```c 
int v[10], *p;
*v = 8;         // stessa cosa che fare v[0] = 8
p = v;          // stessa cosa di p = &v[0]
p = &v[1];      // ora p punta a v[1]
p[1] = 5;       // v[2] = 5;
```
